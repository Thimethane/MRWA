# MRWA Sample Files

This directory contains sample files for testing MRWA workflows.

## Structure

- `research_papers/` - PDF research papers
- `code_repositories/` - Sample code projects
- `links.txt` - Web URLs and YouTube links
- `test_data/` - Test datasets

## Usage

1. **Quick Test**: Use pre-configured templates in web dashboard
2. **Custom Upload**: Add your own files to appropriate directories
3. **Links**: Edit `links.txt` to add custom URLs

## Adding Samples

### Research Papers
Place PDF files in `research_papers/` directory.

### Code Repositories
Create subdirectories in `code_repositories/` for each project.

### Web Links
Add URLs to `links.txt`, one per line. Use # for comments.

## Sample Workflows

- **Research Synthesis**: Uses PDFs + web links
- **Code Analysis**: Uses code repositories
- **Video Analysis**: Uses YouTube links
